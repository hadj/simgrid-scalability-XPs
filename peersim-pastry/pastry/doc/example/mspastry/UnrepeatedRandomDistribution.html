Class UnrepeatedRandomDistribution

public class UnrepeatedRandomDistribution
This initializator assign to the Nodes a nodeId (stored in the protocol MSPastryProtocol) by using a 128-bit (32 byte) random generator, with the bias that no repetitions occurs. In the generator outputs a random number n, the number will NOT be repeated in the output of the current session.
